## Risks identified

### Bugs

1. **Dashboard.tsx:73** — `const isEmpty = !loading && (!totals || totals.income === 0 && totals.expense === 0) && recent.length === 0;` — λόγω operator precedence, αν `totals` υπάρχει αλλά `income === 0` ενώ expense > 0, η isEmpty condition μπορεί να βγει λάθος. Χρειάζεται παρενθέσεις: `(totals.income === 0 && totals.expense === 0)`.

2. **AddTransaction.tsx:56-58** — useEffect dependency `accountId` μπορεί να προκαλεί infinite re-render loop: effect sets accountId → trigger useEffect → re-fetch → set accountId again. Πιθανώς σταματάει λόγω `if (accs.length > 0 && !accountId)` guard, αλλά fragile.

3. **AddTransaction.tsx:44** — `tagName` state αποθηκεύεται αλλά **δεν χρησιμοποιείται** στο `createTransaction` call (line 78-88). Ο tag δεν αποθηκεύεται πραγματικά.

### Security

4. **tauri.conf.json:29** — `"csp": null` — Content Security Policy disabled. Πρέπει να ενεργοποιηθεί πριν production.

5. **tauri.conf.json:53** — `"pubkey": ""` — Updater pubkey κενό. Χωρίς αυτό, auto-updates δεν μπορούν να εγκατασταθούν safely.

### Data integrity

6. **Seed data** — categories count: CLAUDE.md λέει "26 categories", αλλά μετρώντας τα INSERT statements βλέπω 35 categories (6 + 17 + 3 + 9). Minor doc inconsistency.

7. **Book switching** — Dashboard + AddTransaction hardcoded σε `"book-business"`. Προσωπικά transactions δεν μπορούν να προστεθούν/ιδωθούν χωρίς αλλαγή.
